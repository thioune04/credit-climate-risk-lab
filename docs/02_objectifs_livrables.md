# Objectifs & livrables — Credit & Climate Risk Lab

## 1) Objectifs métier
1. Estimer le risque de défaut sur un portefeuille de crédits (PD).
2. Traduire ce risque en impact financier (perte attendue / ECL).
3. Intégrer le risque climatique :
   - physique (exposition géographique aux aléas),
   - transition (scénarios de stress).
4. Produire une restitution décisionnelle via Tableau Public.

## 2) Livrables analytiques (outputs “data”)
- Table PD par prêt (probabilité).
- Table ECL par prêt (euros) et agrégations portefeuille.
- Table exposition climat physique (index par zone).
- Table stress tests (ECL par scénario + delta vs baseline).
- Rapport qualité data (manquants, formats, outliers).

## 3) Livrables décisionnels (outputs “pilotage”)
Tableau Public :
- Dashboard 1 : Portfolio Overview (KPI + segments).
- Dashboard 2 : Physical Exposure (carte + top zones exposées).
- Dashboard 3 : Stress Test (baseline vs scénarios, deltas).

## 4) Livrables projet (professionnalisation)
- Environnement reproductible (Docker + Jupyter).
- Organisation Notebook-first avec sorties exportées.
- Documentation claire + storytelling entretien.
